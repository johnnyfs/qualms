# Wave Collapse

`wave_collapse` is a blank nova-based story seed for coauthoring experiments.

Premise seed: a quiet orbital observatory is watching a collapsed probability wave propagate through old survey beacons. The current file intentionally contains only enough world model to run: a player, one system, one station, and one destination.

Editable story scope:

- Add new systems, orbitals, destinations, objects, NPCs, ships, facts, assertions, and local rules in this top-level story.
- Do not change Core or Nova definitions from the coauthor agent. If a desired behavior needs a new trait, action, relation, or reusable rule in the preludes, report that in feedback.
- Prefer adding content as typed nova entities with clear `Presentable` text and correct `At` or `Path` assertions.
- Keep descriptions durable. Do not say an object, NPC, or ship is currently in a place if it could move later; use assertions and rules for that state.

Current nova content:

- `wave-collapse`: a white dwarf system with no jump hops yet.
- `wave-collapse:needle`: Needle Station, an observation station in the system.
- `wave-collapse:needle:observation-deck`: the starting destination.
